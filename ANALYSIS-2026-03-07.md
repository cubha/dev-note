## 1. 현재 프로젝트 현황

### 1-1. 프로젝트 목적 및 핵심 가치

DevNote는 **NDA 보안 환경에서 일하는 SI 프리랜서를 위한 로컬 접속정보 관리 도구**다.
외부 서버 없이 브라우저 IndexedDB만 사용하는 무설치 웹앱으로, AES-256 암호화로 민감 데이터를 보호한다.

**현재 핵심 가치 (3가지)**:
1. **완전 오프라인** — 외부 API/서버 호출 0
2. **필수 암호화** — 모든 민감 데이터 AES-GCM 암호화
3. **SI 특화** — Server(SSH), DB, API 접속정보 구조화

### 2-2-1. 암호화 완전 제거 결정 근거

**NotePad++ 업그레이드 관점**: 이 앱은 개발자용 정보 정리 도구지, 금고가 아니다.

| 판단 기준 | 분석 |
|----------|------|
| **사용자 기대** | NotePad++에 마스터 비밀번호를 걸고 쓰는 사람은 없음. 정보 정리 도구의 핵심은 빠른 접근 |
| **UX 마찰** | 앱 열 때마다 비밀번호 입력 → 메모앱으로서 치명적 진입 장벽 |
| **코드 복잡도** | 암호화 레이어가 모든 CRUD 경로에 관여 → 불필요한 복잡도 |
| **민감 데이터 책임** | 민감 정보 관리는 사용자 영역. 1Password, Bitwarden 등 전용 도구가 있음 |
| **성능** | 카드 목록 표시 시 N개 직렬 복호화 → 제거 시 즉시 렌더링 |

**DB 스키마 변경 (v8 → 암호화 제거)**:

```typescript
// 현재 (v7): 암호화 분리 구조
items: '++id, folderId, title, *tags, type, order, pinned, updatedAt'
// encryptedContent (Blob), iv (Blob) — 인덱스 외 필드

// 변경 후 (v8): 평문 단일 필드
items: '++id, folderId, title, *tags, type, order, pinned, updatedAt'
// content (string, JSON) — 인덱스 외 필드, encryptedContent/iv 삭제

// 마이그레이션
db.version(8).stores({
  folders: '++id, parentId, name, order',
  items: '++id, folderId, title, *tags, type, order, pinned, updatedAt',
  config: 'id',
}).upgrade(async tx => {
  // encryptedContent → content 평문 변환
  // 기존 암호화 데이터는 복호화 불가 (키 없음) → 빈 content로 초기화
  await tx.table('items').toCollection().modify(item => {
    if (!item.content) {
      item.content = JSON.stringify({ format: 'structured', fields: [] });
    }
    delete item.encryptedContent;
    delete item.iv;
  });
  // config에서 암호화 관련 필드 제거
  await tx.table('config').toCollection().modify(config => {
    delete config.salt;
    delete config.canaryBlock;
    delete config.canaryIv;
  });
});
```

**코드 간소화 효과**:

```
삭제 예상: ~400줄 (crypto.ts ~200줄 + auth/ ~150줄 + 분산 암호화 호출 ~50줄)
간소화: 모든 CRUD 경로에서 async 암복호화 분기 제거 → 동기적 JSON.parse/stringify
성능: 카드 목록 N개 로딩 시 직렬 복호화 O(N) → O(1) 즉시 표시
```

## 3. AI 통합 기술 스택 상세 분석

### 3-1. 왜 Claude API인가 (브라우저 SPA에서 유일한 선택)

| 기능 | Claude API | OpenAI API | Gemini API |
|------|-----------|-----------|-----------|
| **브라우저 CORS 직접 호출** | **공식 지원** | ❌ 프록시 필요 | ❌ 프록시 필요 |
| Structured Outputs (JSON Schema) | GA (100% 보장) | 지원 | 지원 |
| SSE 스트리밍 | 지원 | 지원 | 지원 |
| BYOK SPA 적합성 | **최적** (서버 불필요) | 서버 프록시 필수 | 서버 프록시 필수 |
| 저가형 모델 | Haiku 4.5 ($1/$5) | GPT-4o-mini ($0.15/$0.60) | Flash 2.0 ($0.10/$0.40) |

> **핵심**: OpenAI/Gemini는 브라우저에서 직접 호출 시 CORS 에러 발생.
> Claude API만 `anthropic-dangerous-direct-browser-access` 헤더로 CORS를 해결.
> **서버 없는 SPA에서 외부 LLM API를 호출하려면 Claude가 유일한 선택지.**

### 3-3. Structured Outputs — Smart Paste의 핵심 기술

Claude API의 **Structured Outputs** (2025년 GA)는 **constrained decoding** 기술로
JSON 스키마를 문법(grammar)으로 컴파일한 뒤 토큰 생성 시점에 제한한다.
**모델이 스키마 위반 토큰을 생성하는 것 자체가 불가능**하므로 파싱 에러 걱정 없이 사용 가능.

```typescript
// dev-note Smart Paste 전용 스키마
const SMART_PASTE_SCHEMA = {
  type: "object",
  properties: {
    detectedType: {
      type: "string",
      enum: ["server", "db", "api", "note", "custom"],
      description: "입력 텍스트에서 감지된 카드 타입"
    },
    title: {
      type: "string",
      description: "카드 제목 (자동 생성, 20자 이내)"
    },
    fields: {
      type: "array",
      items: {
        type: "object",
        properties: {
          key:   { type: "string", description: "FIELD_SCHEMAS에 정의된 필드 키" },
          value: { type: "string", description: "추출된 값, 없으면 빈 문자열" }
        },
        required: ["key", "value"],
        additionalProperties: false
      },
      description: "카드 타입에 맞는 필드-값 쌍 배열"
    },
    suggestedTags: {
      type: "array",
      items: { type: "string" },
      description: "추천 태그 (production, staging, mysql 등)"
    },
    confidence: {
      type: "string",
      enum: ["high", "medium", "low"],
      description: "추출 신뢰도"
    }
  },
  required: ["detectedType", "title", "fields", "suggestedTags", "confidence"],
  additionalProperties: false
};
```

### 3-5. API 키 관리 패턴 (BYOK)

```typescript
// store/atoms.ts 추가
export const apiKeyAtom = atom<string | null>(null);

export const aiEnabledAtom = atom((get) => get(apiKeyAtom) !== null);

// sessionStorage 연동 (브라우저 닫으면 자동 소멸)
export const apiKeyPersistAtom = atom(
  () => sessionStorage.getItem('dev-note-claude-key'),
  (_get, set, key: string | null) => {
    if (key) sessionStorage.setItem('dev-note-claude-key', key);
    else sessionStorage.removeItem('dev-note-claude-key');
    set(apiKeyAtom, key);
  }
);
```

**보안 레벨 비교**:

| 저장 방식 | 지속성 | XSS 위험 | 권장 |
|----------|--------|---------|------|
| Jotai atom (메모리) | 새로고침 시 소멸 | 낮음 | 가장 안전, UX 불편 |
| **sessionStorage** | **브라우저 닫으면 소멸** | 중간 | **권장 (보안/UX 균형)** |
| localStorage | 영구 보존 | 높음 | 비권장 |
| IndexedDB (암호화) | 영구 + 암호화 | 중간 | 과도한 복잡도 |

### 4-3. DB 스키마 변경 (v8) — 암호화 제거 + AI 확장

```typescript
// core/db.ts — v8 마이그레이션
db.version(8).stores({
  folders: '++id, parentId, name, order',
  items: '++id, folderId, title, *tags, type, order, pinned, updatedAt',
  config: 'id',
}).upgrade(async tx => {
  // 1. encryptedContent → content 평문 변환
  await tx.table('items').toCollection().modify(item => {
    // 기존 암호화 데이터는 복호화 불가 → 빈 구조화 콘텐츠로 초기화
    if (!item.content) {
      item.content = JSON.stringify({ format: 'structured', fields: [] });
    }
    delete item.encryptedContent;
    delete item.iv;
  });

  // 2. config에서 암호화 관련 필드 제거 + AI 설정 추가
  await tx.table('config').toCollection().modify(config => {
    delete config.salt;
    delete config.canaryBlock;
    delete config.canaryIv;
    config.aiModel = 'claude-haiku-4-5-20241015';
  });
});

// Item 인터페이스 변경
```

### 5-7. Phase 2 — 시맨틱 검색 (로컬 임베딩)

```typescript
// core/embeddings.ts
import { pipeline } from '@huggingface/transformers';

let embedder: Awaited<ReturnType<typeof pipeline>> | null = null;

async function getEmbedder() {
  if (!embedder) {
    embedder = await pipeline(
      'feature-extraction',
      'Xenova/all-MiniLM-L6-v2',
      { dtype: 'q8' }  // 양자화 (~90MB, IndexedDB에 캐시)
    );
  }
  return embedder;
}

export async function generateEmbedding(text: string): Promise<number[]> {
  const model = await getEmbedder();
  const result = await model(text, { pooling: 'mean', normalize: true });
  return Array.from(result.data);
}

export function cosineSimilarity(a: number[], b: number[]): number {
  let dot = 0, normA = 0, normB = 0;
  for (let i = 0; i < a.length; i++) {
    dot += a[i] * b[i];
    normA += a[i] * a[i];
    normB += b[i] * b[i];
  }
  return dot / (Math.sqrt(normA) * Math.sqrt(normB));
}
```

**성능 참고**:
- 모델 다운로드: ~90MB (첫 실행 시, 이후 IndexedDB 캐시)
- 임베딩 생성: ~50ms/텍스트 (WebGPU), ~200ms (WASM)
- 유사도 검색: 1,000개 벡터 대비 ~5ms (순수 JS 순차 비교)

### 8-1. AI Service 모듈 (`core/ai.ts`)

```typescript
import type { ItemType } from './db';

export interface SmartPasteResult {
  detectedType: ItemType;
  title: string;
  fields: Array<{ key: string; value: string }>;
  suggestedTags: string[];
  confidence: 'high' | 'medium' | 'low';
}

export class AIService {
  private apiKey: string;
  private lastCallTime = 0;
  private readonly minIntervalMs = 500;  // 호출 간격 제한

  constructor(apiKey: string) {
    this.apiKey = apiKey;
  }

  async smartPaste(text: string, targetType?: ItemType): Promise<SmartPasteResult> {
    await this.enforceRateLimit();

    const systemPrompt = this.buildSystemPrompt(targetType);
    const response = await this.callClaude({
      model: 'claude-haiku-4-5-20241015',
      max_tokens: 1024,
      system: systemPrompt,
      messages: [{ role: 'user', content: text }],
      output_config: {
        format: { type: 'json_schema', schema: SMART_PASTE_SCHEMA }
      }
    });

    const textBlock = response.content.find(
      (b: { type: string }) => b.type === 'text'
    );
    if (!textBlock) throw new AIError('No text content in response');

    return JSON.parse(textBlock.text) as SmartPasteResult;
  }

  private buildSystemPrompt(targetType?: ItemType): string {
    return `You are a structured data extraction assistant for a developer's tool.
Extract server, database, or API connection information from the given text.

Field mappings by type:
- server: host, port(default 22), username, password, keyPath, note
- db: host, port(default 3306/5432), dbName, username, password, note
- api: url, method(default GET), apiKey, token, headers, note
- note/custom: content

Rules:
- Extract ONLY information explicitly present in the text
- Use empty string "" for fields not mentioned (do not guess)
- Preserve exact values without modification
- Generate a concise title (under 20 chars) describing the resource
- Suggest relevant tags (production, staging, mysql, ssh, etc.)
${targetType ? `- Target type is "${targetType}"` : '- Auto-detect the most appropriate type'}`;
  }

  private async callClaude(body: Record<string, unknown>): Promise<ClaudeResponse> {
    const res = await fetch('https://api.anthropic.com/v1/messages', {
      method: 'POST',
      headers: {
        'x-api-key': this.apiKey,
        'anthropic-version': '2023-06-01',
        'content-type': 'application/json',
        'anthropic-dangerous-direct-browser-access': 'true',
      },
      body: JSON.stringify(body),
    });

    if (res.status === 429) {
      const retryAfter = parseInt(res.headers.get('retry-after') || '60');
      throw new RateLimitError(retryAfter);
    }
    if (res.status === 401) {
      throw new AIError('API 키가 유효하지 않습니다. 설정에서 확인해주세요.');
    }
    if (!res.ok) {
      throw new AIError(`Claude API 오류: ${res.status}`);
    }
    return res.json();
  }

  private async enforceRateLimit(): Promise<void> {
    const elapsed = Date.now() - this.lastCallTime;
    if (elapsed < this.minIntervalMs) {
      await new Promise(r => setTimeout(r, this.minIntervalMs - elapsed));
    }
    this.lastCallTime = Date.now();
  }
}

// 에러 클래스
export class AIError extends Error {
  constructor(message: string) {
    super(message);
    this.name = 'AIError';
  }
}

export class RateLimitError extends AIError {
  retryAfter: number;
  constructor(retryAfter: number) {
    super(`Rate limit exceeded. Retry after ${retryAfter}s`);
    this.retryAfter = retryAfter;
  }
}
```

### 8-2. 정규식 파서 (`core/smart-paste.ts`) — Tier 1 핵심

```typescript
import type { ItemType } from './db';

export interface ParsedField {
  key: string;
  value: string;
  confidence: 'high' | 'medium' | 'low';
}

export interface LocalParseResult {
  detectedType: ItemType | null;
  fields: ParsedField[];
  hasStructuredMatch: boolean;  // Tier 1으로 처리 가능했는지
}

export function detectCardType(text: string): ItemType | null {
  if (/\b(postgres|mysql|mariadb|mongodb|redis|mssql):\/\//i.test(text)) return 'db';
  if (/jdbc:\w+:\/\//i.test(text)) return 'db';
  if (/\b(Database|Initial Catalog|dbname)\s*=/i.test(text)) return 'db';
  if (/\bcurl\s+/i.test(text)) return 'api';
  if (/https?:\/\/.*\b(api|endpoint|webhook)\b/i.test(text)) return 'api';
  if (/\b(api[_-]?key|bearer|authorization)\s*[:=]/i.test(text)) return 'api';
  if (/^\s*Host\s+\S+/m.test(text)) return 'server';
  if (/\w+@[\w.\-]+(?::\d+)?/.test(text)) return 'server';
  if (/\bssh\s+/i.test(text)) return 'server';
  return null;
}

export function localSmartParse(text: string, type: ItemType): LocalParseResult {
  const fields: ParsedField[] = [];
  let hasStructuredMatch = false;

  switch (type) {
    case 'server': hasStructuredMatch = parseServer(text, fields); break;
    case 'db':     hasStructuredMatch = parseDB(text, fields); break;
    case 'api':    hasStructuredMatch = parseAPI(text, fields); break;
    case 'note':
    case 'custom':
      fields.push({ key: 'content', value: text.trim(), confidence: 'high' });
      hasStructuredMatch = true;
      break;
  }

  return { detectedType: type, fields, hasStructuredMatch };
}

function parseServer(text: string, fields: ParsedField[]): boolean {
  // SSH config 블록
  const hostName = text.match(/^\s*HostName\s+(\S+)/m);
  if (hostName) {
    fields.push({ key: 'host', value: hostName[1], confidence: 'high' });
    const m = (p: RegExp) => text.match(p)?.[1];
    if (m(/^\s*User\s+(\S+)/m)) fields.push({ key: 'username', value: m(/^\s*User\s+(\S+)/m)!, confidence: 'high' });
    if (m(/^\s*Port\s+(\d+)/m)) fields.push({ key: 'port', value: m(/^\s*Port\s+(\d+)/m)!, confidence: 'high' });
    if (m(/^\s*IdentityFile\s+(.+)/m)) fields.push({ key: 'keyPath', value: m(/^\s*IdentityFile\s+(.+)/m)!.trim(), confidence: 'high' });
    return true;
  }

  // user@host:port
  const ssh = text.match(/(\w[\w.-]*)@([\w.\-]+)(?::(\d{1,5}))?/);
  if (ssh) {
    fields.push({ key: 'username', value: ssh[1], confidence: 'high' });
    fields.push({ key: 'host', value: ssh[2], confidence: 'high' });
    if (ssh[3]) fields.push({ key: 'port', value: ssh[3], confidence: 'high' });
    return true;
  }

  // 한글/영문 라벨
  return parseLabeledFields(text, fields, [
    ['host', /(?:호스트|host|ip|서버)\s*[:：=]\s*(\S+)/i],
    ['port', /(?:포트|port)\s*[:：=]\s*(\d+)/i],
    ['username', /(?:계정|사용자|유저|아이디|user(?:name)?)\s*[:：=]\s*(\S+)/i],
    ['password', /(?:비밀번호|패스워드|비번|pass(?:word)?)\s*[:：=]\s*(\S+)/i],
  ]);
}

function parseDB(text: string, fields: ParsedField[]): boolean {
  // URI 형식
  const uri = text.match(
    /(postgres(?:ql)?|mysql|mariadb|mongodb|redis|mssql):\/\/(?:([^:@]+)(?::([^@]+))?@)?([\w.\-]+)(?::(\d+))?\/?(\w+)?/i
  );
  if (uri) {
    fields.push({ key: 'host', value: uri[4], confidence: 'high' });
    if (uri[5]) fields.push({ key: 'port', value: uri[5], confidence: 'high' });
    if (uri[6]) fields.push({ key: 'dbName', value: uri[6], confidence: 'high' });
    if (uri[2]) fields.push({ key: 'username', value: uri[2], confidence: 'high' });
    if (uri[3]) fields.push({ key: 'password', value: uri[3], confidence: 'high' });
    return true;
  }

  // Key=Value 형식
  const kvParsed = parseLabeledFields(text, fields, [
    ['host', /(?:Host|Server|Data Source)\s*=\s*([^;\n]+)/i],
    ['port', /Port\s*=\s*(\d+)/i],
    ['dbName', /(?:Database|Initial Catalog|dbname)\s*=\s*([^;\n]+)/i],
    ['username', /(?:User(?:name)?|User Id|UID)\s*=\s*([^;\n]+)/i],
    ['password', /(?:Password|PWD)\s*=\s*([^;\n]+)/i],
  ]);
  if (kvParsed) return true;

  // 한글 라벨
  return parseLabeledFields(text, fields, [
    ['host', /(?:호스트|host|서버)\s*[:：=]\s*(\S+)/i],
    ['port', /(?:포트|port)\s*[:：=]\s*(\d+)/i],
    ['dbName', /(?:데이터베이스|DB명?|database)\s*[:：=]\s*(\S+)/i],
    ['username', /(?:계정|사용자|user(?:name)?)\s*[:：=]\s*(\S+)/i],
    ['password', /(?:비밀번호|패스워드|비번|pass(?:word)?)\s*[:：=]\s*(\S+)/i],
  ]);
}

function parseAPI(text: string, fields: ParsedField[]): boolean {
  let found = false;
  const url = text.match(/https?:\/\/[^\s<>"{}|\\^`\[\]]+/i);
  if (url) { fields.push({ key: 'url', value: url[0], confidence: 'high' }); found = true; }

  const method = text.match(/\b(GET|POST|PUT|PATCH|DELETE)\b/);
  if (method) { fields.push({ key: 'method', value: method[1], confidence: 'medium' }); found = true; }

  const bearer = text.match(/(?:Bearer|Authorization)\s*[:=]?\s*["']?([A-Za-z0-9\-_.~+/=]{20,})["']?/i);
  if (bearer) { fields.push({ key: 'token', value: bearer[1], confidence: 'high' }); found = true; }

  const apiKey = text.match(/(?:api[_-]?key|x-api-key)\s*[:=]\s*["']?([^\s"']+)["']?/i);
  if (apiKey) { fields.push({ key: 'apiKey', value: apiKey[1], confidence: 'high' }); found = true; }

  return found;
}

function parseLabeledFields(
  text: string,
  fields: ParsedField[],
  patterns: [string, RegExp][]
): boolean {
  let found = false;
  for (const [key, pattern] of patterns) {
    if (fields.some(f => f.key === key)) continue;
    const match = text.match(pattern);
    if (match) {
      fields.push({ key, value: match[1].trim(), confidence: 'high' });
      found = true;
    }
  }
  return found;
}
```
